# Анализ юнит-экономики развлекательного приложения: выявление причин стагнации выручки и оптимизация маркетинговых каналов привлечения. 📱
Аналитический проект для развлекательного сервиса. Задача — выявить причины стагнации выручки после запуска спортивного контента и изменения маркетинговой стратегии.

## 🛠 Технологический стек
*   **Python (Pandas, Matplotlib, Seaborn):** Когортный анализ, расчет метрик окупаемости.
*   **Юнит-экономика:** Расчет LTV (Lifetime Value), CAC (Customer Acquisition Cost), ROI/ROMI.
*   **Маркетинговая аналитика:** Анализ каналов привлечения и конверсии в динамике.

## 📈 Ключевые показатели (Метрики)
*   **Retention Rate:** Насколько хорошо новый контент удерживает аудиторию.
*   **CAC by Channel:** Сколько стоит привлечение одного пользователя из разных источников.
*   **LTV (28 days):** Сколько выручки приносит пользователь за критический для модели срок.
*   **ROI:** Окупаемость инвестиций в маркетинг по каждому каналу.

## 📂 Структура репозитория
*   `Анализ системы метрик приложения.ipynb` — основной ноутбук с расчетами и графиками.
*   `conclusions.doc` — итоговые рекомендации по корректировке маркетинговой стратегии.
